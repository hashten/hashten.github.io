Detailed Description

template<typename _Scalar, int _Options>
class Eigen::Map< Sophus::SE3Group< _Scalar >, _Options >

Specialisation of Eigen::Map for SE3GroupBase.

Allows us to wrap SE3 Objects around POD array (e.g. external c style quaternion)
