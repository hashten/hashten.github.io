Detailed Description

template<typename _Scalar, int _Options>
class Eigen::Map< Sophus::SE2Group< _Scalar >, _Options >

Specialisation of Eigen::Map for SE2GroupBase.

Allows us to wrap SE2 Objects around POD array (e.g. external c style complex)
